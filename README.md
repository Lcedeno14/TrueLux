## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The site will be available at `http://localhost:3000`

### Build for Production

To create an optimized production build:

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

To preview the production build locally:

```bash
npm run preview
```

## Sections

4. **About**: Company information and key features
5. **Testimonials**: Client reviews and testimonials

## Customization

### Colors

Edit the CSS custom properties in `styles.css`:

```css
:root {
    --primary: #6366f1;
    --secondary: #ec4899;
    /* ... more colors */
}
```

### Content

All content can be easily modified in `index.html`. Update text, links, and information as needed.

### Images

Replace placeholder project images by modifying the `.project-image` elements in the Projects section. You can add background images or use actual images.
